# Operator Guide

## First Deployment
1. Register operator, repair benches/labs, repair-authority/
   calibration scope, staff and robots.
2. Import existing repair and calibration-certification-record
   history.
3. Run read-only repair-authority-scope and diagnostic/calibration
   robot mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run certification record and audit export.

## Minimum Production Controls
- repair-authority/calibration-scope validation before any repair
  work begins
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (a return-to-
  service release without a completed calibration/post-repair
  inspection, a repair step outside verified repair-authority scope)
- evidence-backed certification records
- audit export for every dispatch, sign-off and release
- backup manual repair-authority review

## Certification
Certified operators must prove robot-safety integrity, repair-
authority discipline, evidence-backed certification records and
human review for release-affecting actions.
